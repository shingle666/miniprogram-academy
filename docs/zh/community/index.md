## 📝 发帖指南

### 提问技巧
1. **明确问题描述** - 详细描述遇到的问题和期望结果
2. **提供代码片段** - 贴出相关的代码，便于他人理解
3. **说明环境信息** - 开发工具版本、基础库版本等
4. **搜索历史帖子** - 避免重复提问，提高效率

### 回答规范
1. **友善耐心** - 对新手保持友善和耐心
2. **准确详细** - 提供准确的解决方案和详细说明
3. **代码示例** - 尽量提供可运行的代码示例
4. **持续跟进** - 关注问题解决情况，及时补充
